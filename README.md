# JSON Redactor - Web Version

Веб-версия программы для удаления строк из JSON файлов.

## Описание

Веб-приложение с теми же функциями, что и десктопная версия:
- Drag and drop для загрузки JSON файлов
- **Удаление нескольких слов одновременно** (от 1 до 5 слов)
- Удаление строк, содержащих любое из указанных слов
- Автоматическое создание резервных копий
- Сохранение исходного форматирования JSON

## Особенности

- **Работает полностью в браузере** - не требует сервера
- **Безопасность** - файлы обрабатываются локально, не отправляются на сервер
- **Современный интерфейс** - темная тема, адаптивный дизайн
- **Drag and Drop** - перетащите файл прямо в окно браузера

## Использование

### Локальный запуск

1. Откройте файл `index.html` в браузере
2. Или используйте локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (если установлен http-server)
npx http-server -p 8000
```

Затем откройте в браузере: `http://localhost:8000`

### Хостинг

Можно загрузить на любой статический хостинг:
- GitHub Pages
- Netlify
- Vercel
- Firebase Hosting
- Или любой другой статический хостинг

## Как использовать

1. Откройте веб-приложение в браузере
2. Перетащите JSON файл в зону загрузки или нажмите для выбора
3. Выберите количество слов для удаления (кнопки 1-5)
4. Введите слова в появившиеся поля (можно ввести от 1 до 5 слов)
5. Нажмите "Обработать файл"
6. Файлы автоматически скачаются:
   - Обработанный файл (с тем же именем)
   - Резервная копия (с датой и временем)

**Примечание:** Удаляются строки, содержащие **любое** из указанных слов.

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript (без зависимостей)
- FileReader API для работы с файлами
- Drag and Drop API

## Примечания

- Все обработка происходит в браузере, файлы не отправляются на сервер
- Резервные копии скачиваются автоматически
- Исходное форматирование JSON сохраняется
- Работает в современных браузерах (Chrome, Firefox, Edge, Safari)
